## Setup

1. Download and install the app for your platform
2. On first launch, enter your User ID and API Key (provided via email)
3. Choose which data to sync
4. Optionally connect your Whoop account

## What does it do?

Rivoo Tracker monitors which apps you use and for how long (similar to RescueTime). Combined with health data from Whoop, we can find correlations between screen time and recovery, sleep and productivity, etc.

Your data is securely stored and only shared within the Rivoo group for combined analysis.
